About Coins Trade World Project

Welcome to the Coins Trade World Project! This website is a comprehensive platform for cryptocurrency enthusiasts, traders, and anyone interested in the world of digital currencies.

Project Overview

The Coins Trade World Project is a part of my second school project at JohnBryce Academy. It aims to provide users with up-to-date information on various cryptocurrencies, including their current prices, historical data, and other relevant details. The project leverages the CoinGecko API to fetch real-time data about a wide range of cryptocurrencies.

Features

  • Search Functionality: Easily search for your favorite coins and get detailed information.
  • Coin Details: View comprehensive details of each coin, including price in multiple currencies, historical data, and more.
  • Favorites: Mark up to 5 of your favorite coins for quick access.

Technology Stack

This project is built using HTML, CSS, and JavaScript. The CoinGecko API is utilized to fetch real-time cryptocurrency data. The project also employs local storage for caching data to enhance performance and reduce the number of API calls.

Future Plans

In the future, I plan to add more features to this project, including more detailed analytics, historical charts, and integration with additional APIs to provide even more comprehensive data.
